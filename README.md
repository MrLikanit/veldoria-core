# Veldoria Core

Кастомный плагин для сервера Veldoria. Добавляет механику добычи спавнеров с RPG-элементами.

## Функционал
- **Специальная кирка:** Не крафтится, выдается админом, имеет защиту от подделки (NBT).
- **Интеграция с AuraSkills:** Шанс добычи зависит от стата Удачи.
- **Система Pity (Жалость):** Каждая неудача повышает шанс следующей попытки на 5%.
- **Поддержка мобов:** Сохраняет тип моба внутри спавнера. Поддерживает пустые спавнеры.
- **Визуализация:** Анимация ритуала перед добычей.

## Команды
- `/veldoria give` - Выдать кирку (нужны права `veldoria.admin`).
